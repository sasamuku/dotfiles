# Understand Context

現在の作業状況とプロジェクトのコンテキストを把握する。

## タスク

`git status` を実行し、次を分析する:

- 現在のブランチ
- 変更済みファイル
- 追跡外ファイル
- ステージ済みの変更

進行中の作業状況をまとめて提示する。
